# The Ascension of Vigil
By RennyC & Community fans

Controls
--------
Keyboard:

Arrow Keys = Movement
Z = Attack
X = Jump
A = Magic
S = Block
Spacebar = Health Flask

Controller:

D-Pad / Left Stick = Movement
X = Attack
A = Jump
B = Magic
Right Trigger = Block
Left Trigger = Health Flask
